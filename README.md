## Installation

1. Install dependencies:
    ```sh
    cd foodclient
    npm install
    ```

2. Install backend server dependencies:
    ```sh
    cd backend
    npm install
    ```
3. Start the backend server:
    ```sh
    node server.js
    ```

4. Start the React development server:
    ```sh
    cd ..
    npm run dev
    ```
